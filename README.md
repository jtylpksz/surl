# sURL
Simple URL Shortener.

![screenshot](https://github.com/jtylpksz/surl/blob/main/images/scr.png?raw=true)

# Setup
Clone the repo and install the dependencies:

```sh
$ git clone https://github.com/martinval11/surl
$ cd surl
$ bun install
```

# Frontend Stack
- Next.js
- Shadcn
- TypeScript
- Turso
